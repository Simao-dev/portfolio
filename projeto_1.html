Treino.app

Aplicativo de treino focado para o personal trainer e o aluno mais avançado, onde é possivel criar treinos personalizados e fazer acompanhamento.

// 01 VISÃO GERAL

O Treino.app é um aplicativo mobile focado em alta performance e autonomia para a gestão de treinos e evolução física. O projeto foi concebido utilizando a metodologia de MVP (Minimum Viable Product) e a abordagem de desenvolvimento offline-first, onde toda a persistência de dados (perfil, rotinas de treino e avaliações) é feita localmente no dispositivo. Isso garante uma aplicação extremamente rápida, que funciona perfeitamente sem internet e reduz os custos iniciais de infraestrutura de nuvem.

// 02 FUNCIONALIDADES

  • Cadastro e Login Convencional
  • Gerenciamento de Perfil
  • Persistência Local de Dados
  • Criação de Rotinas
  • Edição e Customização
  • Exclusão de Treinos
  • Rastreamento de Execução
  • Registro Antropométrico
  • Histórico de Evolução
  • Funcionamento Offline-First
